# 轻量级食堂外卖平台技术文档 python661

## 项目概述

本系统是基于 Python + Django 开发的一体化食堂外卖平台，支持用户端、商家端和管理端的多角色操作。系统集成了菜品管理、订单处理、在线支付、数据分析等功能，适用于高校食堂、企业餐厅及社区餐饮等多种场景。

## 技术栈

- **后端框架：** Django 2.x
- **数据库：** MySQL
- **操作系统兼容性：** Linux/Windows
- **接口设计：** RESTFul + JSON

## 功能模块

### 用户端

- 浏览菜品
- 加购与下单
- 在线支付
- 实时订单状态查看
- 智能推荐（最近热卖、猜你喜欢）

### 商家端

- 店铺信息管理
- 菜品上下架
- 订单语音提醒

### 管理端

- 用户/商家/订单/菜品/评论数据管理
- 日周月销售统计
- 热销排行
- 财务结算

## 系统角色

- 用户：普通消费者
- 商家：店铺运营者
- 管理员：负责整个平台的运营和管理

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x 或更高版本
- 支持Linux和Windows系统

## 部署步骤

1. 安装Python和Django环境
2. 配置MySQL数据库
3. 克隆源码到本地
4. 导入数据库结构
5. 运行服务器
